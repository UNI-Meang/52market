# 오이마켓

오이마켓은 자신의 스토어에서 판매하고 있는 상품을 등록하고 해당 상품을 홍보할 수 있는 <b>Social Networking Service입니다.</b> <br /><br />
사용자는 글과 사진을 <b>업로드</b>하여 자신의 일상을 공유할 수 있고<br />
<b>채팅창</b>을 이용하여 다른 사용자와 즐거운 대화도 나눌 수 있습니다.<br />
<b>팔로우</b> 기능을 사용하면 내가 팔로우한 다른 사용자의 새로운 소식을 홈 화면에서 확인할 수 있고<br />
마음에 드는 피드에 <b>'좋아요'</b> 버튼을 누르거나 <b>댓글</b>을 달 수도 있습니다.<br />
<br />
<b>SNS를 통한 온라인 이웃 되기! 오늘부터 이웃, 오이마켓입니다.</b>

## 7. 개발하며 고민한 부분 및 해결 방법

- `createElement`, `innerHTML`을 이용해 생성된 요소에 접근하기 위해 애를 먹었지만, JavaScript의 이벤트 위임을 통해 해결하였다.
- 동일한 UI에 대한 분기처리 
    - 상단 메뉴, 게시물, 댓글, 상품 판매 더 보기 버튼이 같은 UI이지만 내부 콘텐츠 및 버튼 동작이 달라 분기 처리하면서 함수를 나누어가며 리팩토링하였다.
- 접근성 향상
  - 사용자가 SNS에 업로드 하는 콘텐츠 길이의 예측이 불가하므로 `더 보기`와 `이전` 버튼을 콘텐츠보다 상위에 배치시켜 접근성을 향상시켰다.
  - 사용자가 업로드하는 이미지의 대체 텍스트를 사용자에게 일일이 요청할 경우 사용자의 피로도 증가가 발생할 수 있다고 판단하였다. 때문에 SNS의 경우 해당 사진에 대한 설명이 게시글에 나와 있다고 판단, 사용자가 올린 이미지의 대체 텍스트를 '첨부파일'로 통일시켰다. 
  - 게시물 삭제, 신고 시 처리되었다는 안내 문구가 짧게 노출되도록 하였다.
  - 메뉴/모달창에서 메뉴/모달창 영역 외를 클릭해도 메뉴/모달창을 닫을 수 있도록 하였다.
  - 메뉴/모달창에서 키보드 제어가 잘 되도록 구현하였다.
  - 버튼 요소에 hover, active, focus등의 CSS 효과를 주었다.
